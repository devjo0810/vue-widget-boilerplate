## Project setup

### Quick Setup

* Clone this repository
* `npm install`
* `npm run dev`
* Visit [http://localhost:8080/](http://localhost:8080/)


```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
